GIS Foundations

Geographic Information Systems help us understand places, organize spatial data, and solve real-world problems.

Topics We Will Learn

  • Points, lines, polygons, and boundaries
  • Raster and vector data
  • Coordinate systems and projections
  • QGIS mapping workflows
  • Field data collection
  • Spatial analysis
M1

GIS Foundations: Understanding Place

Focus: What GIS is, why location matters, and how people understand the places around them.

Students will learn:

  • What GIS stands for and what it is used for.
  • The difference between a map and GIS.
  • How location affects jobs, transportation, emergency response, agriculture, history, drones, and communities.
  • Basic geography concepts: location, direction, scale, coordinates, and landmarks.
  • How to describe a familiar place spatially.

Key activities:

  • Create a hand-drawn map of a familiar place.
  • Identify important landmarks and explain why they matter.
  • Practice finding locations using maps and coordinates.
  • Compare different types of maps and geographic information.

Major outputs:

  • “My Place” hand-drawn map.
  • Short written explanation of a familiar place.
  • Basic map-reading activity.
Open Module → M2

GIS Data & Mapping

Focus: Data collection, attributes, cartography, and map layouts.

Students will learn:

  • How GIS represents the real world using points, lines, polygons, and boundaries.
  • What attributes are and how they connect information to map features.
  • How to read and use an attribute table.
  • How to collect local GIS data.
  • Basic map design principles including titles, legends, labels, scale bars, and north arrows.

Key activities:

  • Build a local feature inventory.
  • Collect point data around campus or the community.
  • Create and style a map in QGIS.
  • Turn raw data into a clean map layout.

Major outputs:

  • Field data collection layer.
  • Clean map layout.
  • Local feature dataset.
  • Map design project.
Open Module → M3

Spatial Analysis & Problem Solving

Focus: Using GIS tools to answer questions and solve place-based problems.

Students will learn:

  • How to ask spatial questions such as “What is nearby?” and “What is inside this area?”
  • How to measure distance, area, and proximity.
  • How buffers, selections, overlays, clipping, and joins work.
  • How to compare multiple layers to find relationships.
  • How GIS supports evidence-based decision-making.

Key activities:

  • Create buffers around roads, schools, rivers, or hazards.
  • Use overlays to compare features across multiple layers.
  • Identify possible drone launch sites or community resources.
  • Write a short explanation of GIS findings.

Major outputs:

  • Spatial analysis worksheet.
  • Buffer and overlay map.
  • Local problem-solving analysis.
  • Short written explanation of findings.
Open Module → M4

Drone Mapping, Imagery & Terrain

Focus: Using aerial imagery, drone data, and elevation information in GIS.

Students will learn:

  • The difference between vector and raster data.
  • What orthomosaics, DEMs, DSMs, contours, hillshades, and slope maps are.
  • How drone imagery is processed and used in GIS.
  • How elevation affects planning, drainage, agriculture, construction, and flight operations.
  • How imagery can be interpreted for real-world decision-making.

Key activities:

  • Explore aerial imagery and orthomosaic datasets.
  • Compare drone imagery with roads, buildings, and property boundaries.
  • Create contour, hillshade, or slope visualizations.
  • Identify features and changes visible in imagery.

Major outputs:

  • Drone imagery interpretation map.
  • Elevation or contour map.
  • Terrain analysis activity.
  • Drone mapping project summary.
Open Module → M5

GIS Databases & Data Management

Focus: Organizing GIS data like a professional instead of treating every map layer as a separate file.

Students will learn:

  • Why data organization matters.
  • How tables, records, fields, and IDs work.
  • What unique identifiers and primary keys are.
  • How related tables connect to GIS layers.
  • The purpose of GeoPackages, PostgreSQL, and PostGIS.

Key activities:

  • Clean up poorly organized GIS files.
  • Build a structured GeoPackage.
  • Create related tables using shared IDs.
  • Run basic database and SQL-style queries.

Major outputs:

  • Organized GIS database.
  • Data dictionary.
  • Table relationship diagram.
  • Basic SQL query worksheet.
Open Module → M6

Python & GIS Automation

Focus: Using code to automate repetitive GIS tasks and work with larger datasets.

Students will learn:

  • Why GIS professionals use Python.
  • Basic programming concepts including variables, lists, loops, conditions, and files.
  • How Python can read, filter, clean, and export GIS data.
  • Introductory GeoPandas or PyQGIS workflows.
  • How automation saves time and reduces errors.

Key activities:

  • Open and inspect a CSV or GIS layer with Python.
  • Filter records using simple conditions.
  • Clean and export a dataset.
  • Automate a repetitive GIS task.

Major outputs:

  • Beginner Python notebook or script.
  • Cleaned GIS dataset.
  • Automated export or data-processing workflow.
  • Reflection on how automation improves GIS work.
Open Module → M7

Web GIS, Storytelling & Portfolio

Focus: Sharing GIS work with employers, clients, community members, and decision-makers.

Students will learn:

  • The difference between static maps, web maps, dashboards, and StoryMaps.
  • How to explain GIS findings to non-technical audiences.
  • How to make maps useful for a specific audience.
  • Basic GIS ethics, privacy, and responsible data sharing.
  • How to present GIS work in a portfolio.

Key activities:

  • Turn a previous map into a presentation or story.
  • Create a simple web map or StoryMap.
  • Build a personal GIS portfolio folder.
  • Practice explaining a GIS project to an employer.

Major outputs:

  • Web map, StoryMap, or presentation.
  • GIS portfolio folder.
  • Resume-ready project descriptions.
  • Final presentation.
Open Module → M8

GIS Capstone: Real-World Project

Focus: Applying GIS skills to a complete real-world project from start to finish.

Students will learn:

  • How to define a GIS problem.
  • How to choose appropriate data.
  • How to collect, clean, analyze, and present data.
  • How to work through a professional GIS workflow.
  • How to explain recommendations based on evidence.

Key activities:

  • Select a local or career-connected GIS problem.
  • Develop a project plan and identify needed data.
  • Build a map and complete a spatial analysis.
  • Present findings to classmates, staff, or community partners.

Major outputs:

  • Project proposal.
  • GIS data package.
  • Final map layout.
  • Analysis report.
  • Presentation or StoryMap.
  • Portfolio-ready capstone project.
